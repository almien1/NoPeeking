User guide

Overview

NoPeeking is a text editor that works directly with GPG files. The file is always encrypted while on disk. Once open, a file can be quickly saved back to disk, rendering it unreadable without the password.

Running the program

./nopeeking [filename.gpg]


When a file is first opened, its contents will not be displayed. This is because it's still encrypted. To start editing the file, type its password into the 2nd textbox, and click "Unlock".

At any time, you can stop editing a file by clicking "Lock". This will save the file to disk (encrypted), and remove it from view. To resume editing the file, type the password and click "Unlock".

Use of GPG

Files are saved using GPG's symmetric encryption - you can open the files directly by using gpg -d [filename], and you can create files that are compatible with the text editor by running gpg -c [filename] on a text file

GPG is run in a subprocess. Plaintext and passwords are passed to it using named pipes created in the system's temporary directory.

Timers

An Inactivity timer automatically locks the file (saves it to disk, and removes it from screen) after a period of inactivity. This is the first timer displayed in the status bar, and will reset every time you modify the document

The inactivity timer is designed to keep your file safe if you are distracted and leave the computer while a file is open

A Password timer is slightly different. This timer locks the file if it's been more than a certain length of time since the file's owner proved that they were still in control of the text editor. To reset this timer, type the password and click "Reset".

The inactivity timer is designed to prevent a file being kept open indefinitely by unauthorised parties

Options

Options are specified in the ~/.NoPeeking file, one option per line, in the form option_name = value

font
- The font to use as a Pango description, e.g. "sans bold 10"
fg
- The foreground (text) colour, as either an #800000 RGB value, or a name like black
bg
- The background colour, as either an #800000 RGB value, or a name like white
gpg
- Location of the gpg program
timer1
- Number of seconds for the inactivity timer. Use "min" suffix to specify in minutes
timer2
- Number of seconds for the password timer.
